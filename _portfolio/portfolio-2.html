OptiVest – Portfolio Optimization using Black-Litterman Model

Technologies: Python, NumPy, SciPy | Duration: July 2025

Project Overview

Implemented a sophisticated portfolio optimization system using the Black-Litterman model, combining market equilibrium returns with investor views through a Bayesian framework.

Key Features

  • Black-Litterman Implementation: Combined market equilibrium with investor views using Bayesian methods
  • Multi-Factor Screening: Integrated stock screening across 11 sectors using quantitative signals (market cap, volatility, momentum)
  • Optimization Engine: Solved mean-variance optimization via quadratic programming with sector concentration constraints
  • Performance Metrics: Achieved 23% improvement in risk-adjusted returns and 15% reduction in maximum drawdown relative to benchmark

Technical Implementation

The system demonstrates advanced concepts in modern portfolio theory, including efficient frontier computation, risk parity strategies, and Bayesian inference in financial modeling.
