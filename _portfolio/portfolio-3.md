## Daily Markets Dashboard
**Technologies:** Python, Tkinter, yfinance, FRED API | **Duration:** Feb. 2026

### Project Overview
Built a comprehensive real-time desktop application that aggregates live market data into a unified morning briefing tool for traders and financial professionals.

### Key Features
- **Market Data Integration:** Live data across indices (SPY, QQQ, DIA, IWM), volatility measures (VIX, VVIX), and macro indicators (10Y/2Y Treasury, DXY, crude oil)
- **Economic Calendar:** Real-time earnings announcements and economic event tracking
- **Performance Optimization:** Parallel data fetching pipeline achieving sub-5-second load times
- **Adaptive Refresh:** Intelligent refresh intervals (60s during market hours, 5min premarket)
- **Volatility Analysis:** Implied volatility heat map comparing current IV to 30-day averages across 10 key equities

### Technical Implementation
The application demonstrates efficient API integration, parallel processing for real-time data, and advanced financial market visualization techniques.

### Impact
Provides traders and analysts with a comprehensive market overview in a single interface, reducing the time needed to gather critical market intelligence from multiple sources.
