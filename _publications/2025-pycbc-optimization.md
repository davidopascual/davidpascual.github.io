## Abstract

This research focuses on benchmarking and optimizing the PyCBC gravitational wave detection pipeline specifically for Apple Silicon processors, with emphasis on leveraging the unique architecture of M-series GPUs for scientific computing applications.

## Research Objectives

- Benchmark PyCBC detection pipeline performance on Apple Silicon vs traditional architectures
- Optimize matched-filtering algorithms for M-series GPU architecture
- Analyze memory bandwidth utilization and parallel processing efficiency
- Identify bottlenecks and optimization opportunities in scientific computing workflows

## Technical Approach

The research involves:

### Performance Analysis
- Comprehensive profiling of PyCBC components on M-series hardware
- Comparison of CPU vs GPU implementations
- Memory access pattern optimization
- Parallel processing efficiency measurements

### Algorithm Optimization
- Signal processing workflow improvements
- Matched-filtering algorithm enhancements
- Memory bandwidth optimization techniques
- GPU-specific code optimizations

## Expected Impact

This work contributes to the broader scientific computing community by:
- Providing optimization guidelines for gravitational wave detection on emerging hardware
- Demonstrating the potential of Apple Silicon for high-performance scientific applications
- Contributing to the efficiency of LIGO data analysis pipelines

## Current Status

Research is ongoing as part of a Research Developer position at Syracuse University Department of Physics (December 2025 - Present), focusing on computational aspects of gravitational wave detection and scientific computing optimization.
